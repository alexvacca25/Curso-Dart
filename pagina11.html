Ejemplos Prácticos en Dart

Explora estos 20 ejemplos prácticos para aprender Dart de manera efectiva.

  1. Suma de dos números: int resultado = 5 + 3;
  2. Cálculo del área de un círculo: double radio = 3; double area = pi * radio * radio;
  3. Generación de números aleatorios: import 'dart:math'; int numeroAleatorio = Random().nextInt(100);
  4. Uso de funciones: void saludar(String nombre) { print('Hola, \$nombre'); } saludar('Juan');
  5. Trabajando con listas: List numeros = [1, 2, 3, 4, 5]; print(numeros.length);
  6. Iteración sobre una lista: for (int numero in numeros) { print(numero); }
  7. Operadores lógicos: bool esMayor = (5 > 3); bool esIgual = (5 == 3);
  8. Uso de condicionales: int edad = 18; if (edad >= 18) { print('Eres mayor de edad'); } else { print('Eres menor de edad'); }
  9. Trabajando con mapas: Map persona = {'nombre': 'Juan', 'edad': '30'}; print(persona['nombre']);
  10. Creación de clases: class Persona { String nombre; Persona(this.nombre); }
  11. Uso de clases: Persona persona = Persona('Juan'); print(persona.nombre);
  12. Implementación de interfaces: class Perro implements Animal { void emitirSonido() { print('Guau'); } }
  13. Lectura y escritura de archivos: import 'dart:io'; File archivo = File('datos.txt'); archivo.writeAsStringSync('Hola, mundo');
  14. Manejo de excepciones: try { int resultado = 5 ~/ 0; } catch (e) { print('Error: \$e'); }
  15. Uso de Future para operaciones asíncronas: Future esperar() async { await Future.delayed(Duration(seconds: 2)); print('Operación completada'); }
  16. Consumo de API REST: import 'package:http/http.dart' as http; var response = await http.get(Uri.parse('https://api.example.com/data')); print('Respuesta del servidor: \$response');
  17. Uso de librerías externas: import 'package:crypto/crypto.dart'; String hash = sha256.convert(utf8.encode('password')).toString();
  18. Creación de aplicaciones web con Dart y Flutter: void main() { runApp(MyApp()); }
  19. Desarrollo de aplicaciones móviles: void main() { runApp(MyApp()); }
  20. Implementación de animaciones: import 'package:flutter/material.dart'; AnimationController controller; Animation animation; controller = AnimationController(vsync: this, duration: Duration(seconds: 1)); animation = Tween(begin: 0.0, end: 1.0).animate(controller); controller.forward();
