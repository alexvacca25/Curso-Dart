Clases y Objetos en Dart

Aprende sobre la creación y uso de clases y objetos en Dart.

Definición de Clases

En Dart, puedes definir clases utilizando la palabra clave class. Las clases son plantillas para crear objetos que tienen propiedades y métodos asociados.

// Definición de una clase 'Persona'
class Persona {
  String nombre;
  int edad;

  // Constructor de la clase 'Persona'
  Persona(this.nombre, this.edad);

  // Método para mostrar información de la persona
  void mostrarInformacion() {
    print('Nombre: \$nombre, Edad: \$edad');
  }
}

Creación de Objetos

Para crear un objeto de una clase en Dart, utiliza la palabra clave new seguida del nombre de la clase y los paréntesis.

// Creación de un objeto de la clase 'Persona'
var persona = Persona('Juan', 30);

// Llamando al método 'mostrarInformacion' del objeto 'persona'
persona.mostrarInformacion();

Constructores

En Dart, puedes definir múltiples constructores en una clase utilizando la palabra clave factory o const para crear constructores de fábrica o constantes, respectivamente.

// Constructor de fábrica en la clase 'Persona'
factory Persona.conEdadMinima(String nombre, int edad) {
  if (edad >= 18) {
    return Persona(nombre, edad);
  } else {
    throw ArgumentError('La edad mínima debe ser 18 años.');
  }
}

Herencia

En Dart, puedes crear clases que hereden propiedades y métodos de otra clase utilizando la palabra clave extends.

// Definición de una clase 'Empleado' que hereda de 'Persona'
class Empleado extends Persona {
  String cargo;

  Empleado(String nombre, int edad, this.cargo) : super(nombre, edad);

  @override
  void mostrarInformacion() {
    print('Nombre: \$nombre, Edad: \$edad, Cargo: \$cargo');
  }
}
