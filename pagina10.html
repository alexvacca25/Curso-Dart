Bibliotecas y Paquetes en Dart

Explora cómo utilizar bibliotecas y paquetes en Dart para ampliar la funcionalidad de tu código.

Bibliotecas en Dart

Las bibliotecas en Dart son conjuntos de funciones y clases que puedes importar en tu código para reutilizar funcionalidades. Puedes importar bibliotecas estándar de Dart o crear y usar tus propias bibliotecas.

// Importar una biblioteca estándar de Dart
import 'dart:math';

void main() {
  print('Número aleatorio: \${Random().nextInt(10)}');
}

Paquetes en Dart

Los paquetes en Dart son colecciones de bibliotecas que se pueden compartir y distribuir fácilmente a través del sistema de gestión de paquetes de Dart, llamado Pub. Puedes utilizar paquetes de Dart para agregar funcionalidades adicionales a tu proyecto.

// Ejemplo de archivo pubspec.yaml que especifica dependencias del paquete
dependencies:
  http: ^0.13.3

// Importar una biblioteca de un paquete
import 'package:http/http.dart' as http;

void main() async {
  var response = await http.get(Uri.parse('https://api.example.com/data'));
  print('Respuesta del servidor: \${response.statusCode}');
}

Uso de Paquetes Externos

Además de los paquetes oficiales proporcionados por Dart, puedes utilizar paquetes externos creados por la comunidad. Estos paquetes se pueden encontrar en el sitio web de Dart Pub y se pueden agregar a tu proyecto fácilmente.

// Ejemplo de archivo pubspec.yaml que especifica una dependencia de un paquete externo
dependencies:
  flutter:
    sdk: flutter
  http: ^0.13.3

// Importar una biblioteca de un paquete externo
import 'package:http/http.dart' as http;

void main() async {
  var response = await http.get(Uri.parse('https://api.example.com/data'));
  print('Respuesta del servidor: \${response.statusCode}');
}
