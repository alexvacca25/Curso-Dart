Programación Asíncrona en Dart

Aprende sobre la programación asíncrona y manejo de tareas en Dart.

Asincronía en Dart

Dart ofrece soporte para programación asíncrona utilizando la palabra clave async y await. Esto permite realizar operaciones sin bloquear el hilo principal del programa.

// Función asincrónica que simula una operación de espera
Future esperar() async {
  await Future.delayed(Duration(seconds: 2));
  print('Operación completada');
}

// Llamando a la función asincrónica
void main() {
  print('Iniciando operación');
  esperar();
  print('Continuando con otras tareas');
}

Manejo de Futures

Los Futures en Dart representan un valor que puede estar disponible en el futuro o puede fallar con un error. Puedes esperar que un Future se complete utilizando await o manejar el resultado utilizando .then() y .catchError().

// Función asincrónica que devuelve un Future
Future obtenerNumero() async {
  await Future.delayed(Duration(seconds: 2));
  return 42;
}

// Manejo del Future utilizando await
void main() async {
  var resultado = await obtenerNumero();
  print('El número obtenido es: \$resultado');
}

Programación Concurrente

Dart también ofrece soporte para programación concurrente mediante el uso de Isolate. Los Isolates son procesos independientes que se ejecutan en su propio hilo, lo que permite realizar tareas en paralelo.

// Ejemplo de creación de un Isolate
import 'dart:isolate';

void tarea(SendPort sendPort) {
  sendPort.send('Mensaje desde el Isolate');
}

void main() async {
  ReceivePort receivePort = ReceivePort();
  Isolate isolate = await Isolate.spawn(tarea, receivePort.sendPort);
  receivePort.listen((message) {
    print('Mensaje recibido: \$message');
  });
}
