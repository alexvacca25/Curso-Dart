Manejo de Errores en Dart

Aprende sobre cómo manejar errores y excepciones en Dart.

Excepciones en Dart

En Dart, las excepciones se utilizan para manejar errores que ocurren durante la ejecución de un programa. Puedes lanzar una excepción utilizando la palabra clave throw y manejarla utilizando try, catch y finally.

// Función que lanza una excepción si el argumento es nulo
void validarArgumento(String? argumento) {
  if (argumento == null) {
    throw ArgumentError('El argumento no puede ser nulo');
  }
}

void main() {
  try {
    validarArgumento(null);
  } catch (e) {
    print('Se produjo un error: \$e');
  } finally {
    print('Continuando con otras tareas');
  }
}

Tipos de Excepciones

Dart proporciona varios tipos de excepciones predefinidas, como ArgumentError, FormatException, RangeError, entre otros. Puedes utilizar estos tipos de excepciones o crear tus propias excepciones personalizadas.

// Función que lanza una excepción personalizada
void lanzarExcepcionPersonalizada() {
  throw FormatException('Se produjo un error de formato');
}

void main() {
  try {
    lanzarExcepcionPersonalizada();
  } catch (e) {
    print('Se produjo un error: \$e');
  }
}

Manejo de Errores Asíncronos

Cuando manejas errores en operaciones asíncronas, puedes utilizar catchError o onError para capturar errores que ocurran en un Future o un Stream.

// Función asíncrona que simula una operación de espera
Future esperar() async {
  throw Exception('Se produjo un error durante la espera');
}

void main() {
  esperar().catchError((e) {
    print('Se produjo un error: \$e');
  });
}
